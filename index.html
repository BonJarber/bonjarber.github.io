What I do

I help organizations improve the security of their web applications through more than a bland report. I've done both professional penetration testing and internal security engineering. Now I share this experience with organizations like yours.

Working with me will give you a more personal connection than you get with most large consulting companies. I can penetration test your applications, perform security reviews of your source code and system designs, and host talks or workshops.

Security Reviews

Web Application Penetration Test

Using a mixture of manual testing and custom automation, I'll test your web application for a wide range of security vulnerabilities such as XSS, SQLi, CSRF, logic and auth flaws, sensitive data exposure, and more. These engagements can range from fully remote blackbox tests to deep dives into the code base shoulder-to-shoulder next to your engineers. I'm flexible to the needs of your organization.

Source Code and System Design Review

I'll work with your engineers to identify flaws and weaknesses in the design and implementation of your application. This type of review is best for recognizing large architectural problems, logic errors, crytographic mistakes, and key management issues. I'll recommend both specific fixes as well as general practice improvements.

Training and Workshops

For Developers

I teach developers how malicious individuals are going to attack their code, what they can do to write more secure code, and how to think like an attacker when designing new systems. These talks and trainings can be customized for your specific tech stack or industry.

For Security Engineers

I do onsite web application security testing methodology and tooling training for junior security engineers and developers interested in security. This can also include how to do secure code review, threat modeling, and security automation.
